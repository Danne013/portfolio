Proyectos Académicos

Sistema de nóminas

Aplicación de escritorio que simula el sistema de nómina de una empresa ficticia, permite a los empleados consultar su recibo de pago de forma ordenada.

El sistema es administrado por un usuario con control total sobre la gestión de empleados, departamentos y puestos. Este usuario también aplica las percepciones y deducciones para el cálculo de la nómina.

Proyecto realizado de manera colaborativa.

Windows Forms C Sharp SQL

DouDou!

Plataforma web tipo blog pensada como un espacio para coleccionistas de figuras, donde los usuarios pueden compartir publicaciones y interactuar con ellas mediante comentarios.

La plataforma cuenta con filtros que facilitan la búsqueda de contenido, esto permite a los usuarios encontrar publicaciones de forma más eficiente.

Proyecto realizado de manera colaborativa.

HTML CSS Bootstrap Java SQL

Gato Futbolero

Aplicación web de tipo red social con temática de los mundiales de la FIFA, enfocada en la interacción entre usuarios a través de contenido relacionado con estos torneos.

Fue diseñada para estar accesible a todo público, sin embargo, los usuarios deben contar con una cuenta para poder interactuar mediante publicaciones, likes, comentarios y visualizaciones, mientras que el administrador se encarga de la gestión de infografías, moderación de publicaciones y administración de usuarios.

Proyecto realizado de manera colaborativa.
